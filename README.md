# 雪球组合调仓监控脚本
该脚本可用于监控雪球组合的变动情况，现在只监控yyb凌波的“可转债轮动策略”这一个组合。一旦组合调仓，会自动触发一条短信提示。  
该脚本只看源代码，不涉及json请求。  
如遇无法输出结果，可尝试更新headers中的Cookies值，取值来源：右键-检查-Network-XHR-web?category=web_behavior
